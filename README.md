# Trello Clone

[![Deploy to GitHub Pages](https://github.com/LostFeniks/trello-clone/actions/workflows/deploy.yml/badge.svg)](https://github.com/LostFeniks/trello-clone/actions/workflows/deploy.yml)

Trello-like task management system built with React, TypeScript, and Webpack.

## Features

- ✅ Create, delete, and move cards between columns
- ✅ Drag and drop support with visual feedback
- ✅ Persistent storage using localStorage
- ✅ Responsive design
- ✅ Three fixed columns: To Do, In Progress, Done

## Demo

[View Live Demo](https://LostFeniks.github.io/trello-clone/)

## Technologies

- React 18
- TypeScript
- Webpack 5
- CSS Modules
- localStorage for persistence

## Local Development

### Prerequisites

- Node.js (v18 or higher)
- Yarn
